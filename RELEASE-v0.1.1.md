## 这一次，我们解决了什么问题？

kali-claw v0.1.0 已经是一个攻击技术覆盖全面的渗透测试智能体——25 个技能领域横跨 OWASP Top 10、网络渗透、云安全、移动安全、无线安全等方向。

但在实际渗透测试工作中，仅有攻击技术是不够的。我们发现了一个关键缺口：

> **技术能力强，但流程能力弱。**

具体来说：

- 面对一个新披露的 CVE，只能搜索，无法系统化地产出深度分析报告
- 扫出了漏洞，但无法从"发现"推进到"构造 PoC 并提交"
- 每次渗透测试的操作缺乏规范化证据链，难以审计和重现
- 只会黑盒测试，面对源代码无从下手

v0.1.1 正是为解决这四个问题而来。

## 六大新技能一览

### 1. Deep Research — 多源情报研究

**一句话：像安全研究员一样做调研。**

六阶段研究流程：定义范围 → 规划搜索 → 多源检索 → 深度阅读 → 交叉验证 → 综合报告。每一个结论必须有来源引用。

与 OSINT 的区别：OSINT 回答"目标有什么"，Deep Research 回答"这件事的来龙去脉"。

**典型场景：** 一个新 CVE 披露后，快速产出包含影响范围、利用状态、补丁情况、检测规则的完整分析报告。

---

### 2. Security Bounty Hunter — 漏洞赏金猎手

**一句话：从"发现漏洞"到"拿到赏金"的完整路径。**

聚焦远程可达、用户可控、有实际影响的高价值漏洞。明确界定值得报告的漏洞模式（SSRF、认证绕过、反序列化 RCE、命令注入等），过滤掉自 XSS、缺少安全头等低信号噪音。提供标准化的报告结构。

**典型场景：** 漏洞评估扫出 10 个 SQL 注入点，赏金猎手筛选出 3 个远程可达的，逐个构造最小 PoC 并撰写可提交的报告。

---

### 3. Terminal Ops — 证据链终端操作

**一句话：每一步操作都可追溯、可验证、可重现。**

五阶段执行流程：确认目标 → 读取失败面 → 带证据执行 → 窄幅变更 → 精确状态报告。每次操作记录时间戳、命令、输出文件和执行状态。

这是一个**横切技能**——不是教你攻击什么，而是教你如何规范地执行任何攻击操作。

**典型场景：** 红队演练中，所有 nmap 扫描、漏洞利用、后渗透操作都按证据链协议记录，最终产出可直接用于渗透测试报告的证据文件。

---

### 4. Search First — 先搜索，再动手

**一句话：别重复造轮子。**

四级决策矩阵：直接使用 → 修改适配 → 组合利用 → 自行开发。搜索源覆盖 Exploit-DB、GitHub、Metasploit、Nuclei 模板、HackTricks、GTFOBins 等。

同样是一个**横切技能**——在执行任何任务前，先检查是否有现成的工具或利用代码。

**典型场景：** 面对 Apache 某版本的路径遍历漏洞，先搜索发现已有多个 PoC，直接用 curl 命令验证，而非从零开始手工测试。

---

### 5. Security Review — 安全审计清单

**一句话：像高级安全顾问一样做系统审计。**

覆盖 OWASP Top 10 全部类别的结构化检查清单：密钥管理、输入验证、注入缺陷、认证授权、安全头、API 安全、依赖安全。五级严重性分类（Critical/High/Medium/Low/Info），七步审计流程。

**典型场景：** 白盒渗透测试中，按清单逐项审计目标应用，在"注入缺陷"环节调用 web-sqli、web-xss 等专项技能深入测试。

---

### 6. Repo Scan — 源码资产审计

**一句话：先搞清楚代码库里有什么，再决定审哪里。**

四级模块判定体系：核心资产（深度审计）/ 提取更新（替换为包管理器版本）/ 重建（重构）/ 废弃（删除减面）。跨技术栈覆盖 C/C++、Java、Web、Python、Go、Rust、PHP。自动检测嵌入式第三方库和已知 CVE。

**典型场景：** 审计一个 5 万行代码的 C++ 项目，发现嵌入了一个 2015 年版本的 FFmpeg，立即标记为高优先级安全风险。

## 四大能力闭环

六个新技能不是简单的数量叠加，而是构建了四个关键能力闭环：

```
情报闭环：OSINT 工具操作 → Deep Research 情报综合研究
变现闭环：漏洞扫描清单 → Security Bounty Hunter PoC + 报告
流程闭环：自由操作 → Terminal Ops 证据链 + Search First 效率决策
白盒闭环：纯黑盒 → Repo Scan 资产分类 + Security Review 系统审计
```

**v0.1.0 是一个攻击技术全面的渗透测试工具库。**

**v0.1.1 是一个从侦察到报告全流程覆盖的渗透测试智能体。**

## 技能协同示例

### Bug Bounty 完整流程

```
osint（收集目标资产）
  → deep-research（深入研究技术栈漏洞生态）
    → vulnerability-assessment（自动化扫描）
      → security-bounty-hunter（筛选 + PoC + 提交报告）
```

### 白盒渗透测试

```
repo-scan（代码资产分类，识别第三方库和风险模块）
  → security-review（OWASP Top 10 系统审计）
    → web-sqli / web-xss（针对发现深入利用）
```

### 新 CVE 响应

```
search-first（搜索已有 PoC 和利用代码）
  → deep-research（深入研究影响范围和检测方法）
    → terminal-ops（带证据链验证环境复现）
```

### 红队演练

```
terminal-ops（全程按证据链协议执行）
  → recon-osint（目标侦察）
    → network-pentest（网络渗透）
      → post-exploitation（后渗透）
```
